Данный проект является решением "Тестового задания по Web+Java"

Используемые технологии:

HTML, CSS, JavaScript(+jQuery) - на клиентской стороне
SprinMVC, Hibernate - на серверной стороне
Используемая СУБД - MySQL
Build tool - Maven

Для запуска приложения нужно указать свойства jdbc.username и jdbc.password в файле hibernate.properties.
